# Задача

Написать PCI-драйвер с probe и release для сетевого адаптера. 
Вставить во функцию probe инициализацию сетевого драйвера и регистрацию устройства. 
Реализовать чтение MAC-адреса.

# Выполнение

1. Соберём модуль командой make
2. Узнаём PCI-устройство (lspci -nn | grep -i ethernet)
3. Проверяем, какой драйвер сейчас управляет устройством (lspci -k -s 00:03.0)
4. Временно выгружаем e1000e (sudo modprobe -r e1000e)
5. Проверяем, что устройство теперь свободно (lspci -k -s 00:03.0)
6. Загружаем наш драйвер (sudo insmod pci_net_xmpl.ko)
7. Проверим кольцеовй буфер (dmesg | tail -n 10)
8. Смотрим список интерфейсов, есть ли там наш (ip link show)
9. Поднимаем интерфейс (sudo ip link set pci_xmpl0 up)
10. Добавляем адрес (sudo ip addr add 10.0.1.1/24 dev pci_xmpl0)
11. Отправляем тестовый пакет (ping -c 1 10.0.1.2)
12. Смотрим кольцевой буфер (sudo dmesg | tail)
13. Выгружаем модуль (sudo rmmod pci_net_xmpl)
14. Возвращаем стандартный (sudo modprobe e1000e)
15. Перезапускаем систему (sudo reboot)
